# Vue3 PC 管理后台

## 技术栈
- Vue 3.4
- Vue Router 4
- Pinia 状态管理
- Element Plus UI 组件库
- Axios HTTP 客户端

## 安装依赖
```bash
npm install
```

## 开发环境
```bash
npm run dev
```
访问: http://localhost:3000

## 构建生产
```bash
npm run build
```

## 功能模块
- ✅ 用户登录/登出
- ✅ 项目管理
- ✅ 模板管理（动态表单配置）
- ⏳ 点位管理（地图中心）
- ⏳ 审核管理
- ⏳ 数据统计看板

## 目录结构
```
src/
├── api/          # API 接口
├── views/        # 页面组件
├── components/   # 通用组件
├── router/       # 路由配置
├── store/        # 状态管理
└── App.vue
```
