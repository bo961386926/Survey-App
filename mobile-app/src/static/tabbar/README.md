# TabBar 图标资源说明

需要准备以下图标文件(建议尺寸: 81x81px):

## 图标列表

1. **home.png / home-active.png** - 工作台图标
   - 普通状态: 灰色房子图标
   - 激活状态: 蓝色房子图标

2. **point.png / point-active.png** - 点位图标
   - 普通状态: 灰色定位图标
   - 激活状态: 蓝色定位图标

3. **map.png / map-active.png** - 地图图标
   - 普通状态: 灰色地图图标
   - 激活状态: 蓝色地图图标

4. **survey.png / survey-active.png** - 勘查图标
   - 普通状态: 灰色文档图标
   - 激活状态: 蓝色文档图标

5. **my.png / my-active.png** - 我的图标
   - 普通状态: 灰色用户图标
   - 激活状态: 蓝色用户图标

## 图标要求

- 格式: PNG(支持透明背景)
- 尺寸: 81x81px (推荐)
- 颜色: 
  - 普通状态: #94A3B8 (灰色)
  - 激活状态: #2563EB (蓝色)

## 图标生成建议

可以使用以下工具生成图标:
- IconFont (https://www.iconfont.cn/)
- Flaticon (https://www.flaticon.com/)
- Material Icons (https://fonts.google.com/icons)

搜索关键词:
- home, house, dashboard
- location, pin, marker
- map, navigation
- document, file, clipboard
- person, user, profile
