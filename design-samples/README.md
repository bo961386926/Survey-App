# 青泓项目勘察信息采集与审核系统 - UI设计小样

## 概述

本文件夹包含青泓项目勘察信息采集与审核系统的UI设计小样，采用现代亮色主题设计，参考UI UX Pro Max样例风格，注重用户体验和视觉一致性。

## 设计原则

- **亮色主题**: 采用白色背景，蓝色主色调，适合长时间办公使用
- **现代美学**: 使用渐变、毛玻璃效果、微动画等现代设计元素
- **响应式设计**: 支持PC和移动端自适应布局
- **一致性**: 统一的色彩系统、字体和组件样式

## 色彩系统

- 主色: #2563EB (Trust Blue)
- 强调色: #F97316 (Orange)
- 成功: #10B981 (Emerald)
- 警告: #F59E0B (Amber)
- 危险: #EF4444 (Red)
- 信息: #8B5CF6 (Violet)
- 背景: #FFFFFF (白色)

## 设计小样列表

### 1. dashboard-v2.html - 工作台仪表板
- **平台**: PC端
- **功能**: 系统首页，显示统计数据、快捷操作、最近活动
- **特色**: 毛玻璃导航栏、数据卡片、图表展示

### 2. dashboard-analytics.html - 勘察数据仪表板
- **平台**: PC端
- **功能**: 实时监控勘察数据、项目进度、数据质量指标
- **特色**: 参考UI UX Pro Max样例风格，现代化数据可视化界面

### 3. collection-mobile-v2.html - 移动端现场采集
- **平台**: 移动端
- **功能**: 现场数据采集界面，支持GPS定位、照片拍摄、表单填写
- **特色**: 适配移动设备，安全区域适配，触摸友好

### 4. point-list-v2.html - 点位管理列表
- **平台**: PC端
- **功能**: 勘察点位列表展示、筛选、排序、批量操作
- **特色**: 数据表格、筛选面板、操作按钮组

### 5. review-detail.html - 审核详情页面
- **平台**: PC端
- **功能**: 勘察数据审核详情，包含原始数据、审核意见、审批流程
- **特色**: 详情布局、状态指示器、操作历史

### 6. user-management.html - 人员管理页面
- **平台**: PC端
- **功能**: 勘察团队成员管理，包含成员信息、角色权限、统计数据
- **特色**: 成员卡片布局、筛选搜索、状态管理、权限编辑

### 7. project-management.html - 项目管理页面
- **平台**: PC端
- **功能**: 项目创建和管理，包含项目列表、进度跟踪、团队分配
- **特色**: 项目卡片网格、搜索过滤、进度条、状态指示

## 技术实现

- **HTML5 + CSS3**: 现代Web标准，无框架依赖
- **响应式布局**: Flexbox和Grid布局
- **动画效果**: CSS动画和变换
- **设计风格**: 参考UI UX Pro Max样例，采用现代亮色主题
- **字体**: Inter + Noto Sans SC，优化中英文字体渲染
