sequenceDiagram autonumber participant C as 客户端 participant F as JwtAuthenticationFilter participant J as JWT Service participant U as UserDetailsService participant D as 数据库 participant S as SecurityContext C->>F: HTTP请求 + Bearer Token F->>J: 解析验证 Token alt Token无效/过期 J-->>F: 验证失败 F-->>C: 401 Unauthorized else Token有效 J-->>F: {username, loginType} alt loginType = "collab" (协作入口) F->>F: CollabSecurityService Note over F: 黑名单过滤
仅允许GET白名单
记录collab_access_log F->>S: ROLE_COLLAB else loginType = "internal" (内部用户) F->>U: loadUserByUsername(username) U->>D: 查询用户+角色+权限 D-->>U: User + Roles + Permissions U->>U: 通配符"*"展开 U-->>F: UserDetails(GrantedAuthorities) F->>S: UsernamePasswordAuthToken end F-->>C: 放行到Controller end

📋权限码自动注册与同步机制

sequenceDiagram autonumber participant App as 应用启动 participant BP as BeanPostProcessor participant PR as PermissionRegistry participant PI as PermissionInitializer participant DB as sys_permission表 App->>BP: 扫描所有 @RestController BP->>PR: 提取 @PreAuthorize 注解 Note over PR: hasAuthority('point:view')
hasAnyAuthority('audit:pass','audit:reject') PR->>PR: 注册到 ALL_PERMISSIONS 集合 Note over PR: {point:view, audit:pass,
audit:reject, project:edit, ...} App->>PI: ApplicationReadyEvent 触发 PI->>DB: 查询现有权限码 DB-->>PI: 返回已存在权限码列表 loop 遍历 ALL_PERMISSIONS alt 权限码不存在 PI->>DB: INSERT INTO sys_permission else 权限码已废弃 PI->>DB: UPDATE SET status = 0 else 权限码已存在 PI->>PI: 跳过 end end PI-->>App: 权限同步完成

日志类型 记录内容 风险等级 存储表

风控拦截 越权访问、异常请求 高 operation_log
纠偏操作 坐标修改前后值、修改人、偏移距离 中 location_correction_log
配置变更 系统参数修改、字典变更 中 operation_log
