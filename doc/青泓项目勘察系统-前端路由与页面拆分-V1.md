# 1. 总体说明

本文档用于前端研发快速建立项目结构，定义：
- 路由层级
- 页面职责
- 核心组件拆分
- 建议状态管理范围

技术约束：
- PC 后台：Vue3 + Vue Router + Pinia + Element Plus
- 移动端：uni-app + Pinia + uView Plus

# 2. PC 后台路由表

## 2.1 一级路由
- `/login`
- `/dashboard`
- `/projects`
- `/projects/:id`
- `/points/map`
- `/templates`
- `/templates/:id`
- `/audit`
- `/audit/:resultId`
- `/exports`
- `/collab`
- `/users`
- `/logs`
- `/messages`

## 2.2 详细路由

### `/login`
- 页面：登录页
- 说明：后台账号登录、验证码登录

### `/dashboard`
- 页面：首页工作台
- 说明：项目总览、待审核、地图摘要、消息提醒

### `/projects`
- 页面：项目管理页
- 说明：项目列表、项目筛选、项目创建

### `/projects/:id`
- 页面：项目详情/标段进度页
- 说明：项目看板、标段进度、成员、风险提示

### `/points/map`
- 页面：点位地图中心
- 说明：点位地图、列表联动、批量派发

### `/templates`
- 页面：模板列表页
- 说明：模板列表、版本摘要、创建模板

### `/templates/:id`
- 页面：动态模板设计器
- 说明：字段库、画布、属性、联动规则

### `/audit`
- 页面：审核列表页
- 说明：待审核队列、筛选、批量通过

### `/audit/:resultId`
- 页面：审核详情与版本对比页
- 说明：图片校验、地图校验、表单详情、版本差异

### `/exports`
- 页面：导出与报表中心
- 说明：导出配置、任务列表、报表预览

### `/collab`
- 页面：第三方协作与权限配置页
- 说明：协作入口、权限边界、风控日志

### `/users`
- 页面：用户与角色管理页
- 说明：用户列表、角色列表、权限明细

### `/logs`
- 页面：操作日志中心
- 说明：操作日志、风险日志、趋势图

### `/messages`
- 页面：消息与任务提醒
- 说明：任务流、通知流、推荐处理顺序

- 提交审核

- 恢复草稿
- 删除草稿

# 6. 状态管理建议

## 6.1 Pinia Store 建议
- `useAuthStore`
- `useProjectStore`
- `usePointStore`
- `useTemplateStore`
- `useAuditStore`
- `useExportStore`
- `useCollabStore`
- `useUserStore`
- `useLogStore`
- `useNotificationStore`

## 6.2 每个 Store 职责

### `useAuthStore`
- 用户信息
- token
- 权限码
- 当前项目范围

### `useProjectStore`
- 项目列表缓存
- 当前项目详情
- 标段摘要

### `usePointStore`
- 地图筛选条件
- 点位列表
- 点位选中态

### `useTemplateStore`
- 当前模板草稿
- 当前字段选中态
- 规则编辑态

### `useAuditStore`
- 待审核列表
- 审核详情
- 版本 diff

### `useNotificationStore`
- 未读数
- 消息列表
- 当前分类筛选

# 8. 联调注意事项

1. 所有状态字段统一使用枚举，不允许页面自行写死中文判断
2. 审核与提交接口需保证幂等
3. 模板字段 JSON 必须有版本号
4. 第三方权限要后端强校验，前端仅做展示限制
5. 导出任务需轮询或 websocket 订阅状态
