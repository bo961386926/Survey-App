# 1. 项目概述

## 1.1 项目名称
青泓项目勘察信息采集与审核系统

## 1.2 建设目标
建设一套覆盖 `项目管理`、`点位管理`、`移动采集`、`审核闭环`、`动态模板`、`导出归档`、`第三方协作` 的勘察业务系统，解决以下问题：
- 现场采集标准不统一
- 不同排口类型字段差异大，固定表单无法适配
- 弱网环境下数据易丢失
- 定位偏差缺少统一纠偏机制
- 审核缺少版本留痕与驳回闭环
- 第三方协作入口不安全、权限不可控

## 1.3 建设范围
- PC 后台管理端：Vue3 + Element Plus
- 移动采集端：uni-app + uView Plus
- 第三方协作入口：基于 uni-app / 小程序受限入口
- 服务端：RESTful API
- 数据库：MySQL 8.0
- 文件存储：OSS
- 地图服务：高德地图 / 微信小程序原生 map

## 1.4 核心价值
- 实现标准化采集
- 实现模板动态配置
- 实现离线草稿与弱网恢复
- 实现地图纠偏与位置追溯
- 实现审核闭环与版本管理
- 实现导出归档与审计留痕
- 实现第三方协作安全接入

## 1.5 项目成功指标
- 草稿丢失率小于 0.5%
- 驳回重提版本 100% 可追溯
- PDF 报告生成成功率大于 99%
- 第三方越权访问 100% 被阻断并记录
- 审核记录 100% 保留操作链路

# 2. 角色定义

## 2.1 角色列表
- 管理员
- 项目负责人
- 审核员
- 采集员
- 第三方协作人员

## 2.2 角色职责

### 管理员
- 创建项目、标段、点位
- 配置模板、发布模板版本
- 配置用户、角色、权限
- 创建/撤销第三方协作入口
- 执行全局导出
- 查看操作日志与风控日志

### 项目负责人
- 管理所属项目
- 配置项目成员
- 绑定模板与标段
- 查看项目推进、审核压力、导出情况
- 创建项目级协作入口

### 审核员
- 查看待审核列表
- 审核采集结果
- 查看版本差异
- 生成 PDF 报告
- 查看审核相关日志

### 采集员
- 查看分配点位
- 地图定位与人工纠偏
- 填写表单
- 拍照上传
- 保存草稿
- 提交审核

### 第三方协作人员
- 通过授权入口进入系统
- 查看授权范围内点位
- 协助填写、补充照片
- 不可审核、不可删除、不可全局导出

# 4. 业务流程

## 4.1 主业务流程
1. 管理员创建项目
2. 配置成员、标段、模板
3. 导入/新增点位
4. 点位分配给采集员
5. 采集员现场定位、拍照、填表、保存草稿
6. 采集员提交审核
7. 系统生成新版本并进入待审核
8. 审核员执行通过或驳回
9. 驳回后采集员修改重提
10. 审核通过后生成 PDF、纳入导出与归档

## 4.2 模板流程
1. 创建模板草稿
2. 配置字段、校验规则、联动规则
3. 预览移动端效果
4. 发布模板版本
5. 绑定项目/标段
6. 后续调整必须生成新版本

## 4.3 第三方协作流程
1. 创建协作入口
2. 设置授权点位范围、权限、有效期
3. 生成链接/小程序码/临时账号
4. 第三方在授权范围内执行查看/协助填报
5. 系统记录访问和操作日志
6. 到期自动失效或手工撤销

# 5. 状态机

## 5.1 项目状态
- 草稿
- 进行中
- 已暂停
- 已完成
- 已归档

流转：
- 草稿 -> 进行中
- 进行中 -> 已暂停
- 已暂停 -> 进行中
- 进行中 -> 已完成
- 已完成 -> 已归档

## 5.2 点位状态
- 待分配
- 待采集
- 草稿中
- 待审核
- 审核通过
- 驳回待修改
- 已归档
- 作废

流转：
- 待分配 -> 待采集
- 待采集 -> 草稿中
- 草稿中 -> 待审核
- 待审核 -> 审核通过
- 待审核 -> 驳回待修改
- 驳回待修改 -> 草稿中 / 待审核
- 审核通过 -> 已归档
- 任意有效状态 -> 作废

## 5.3 采集结果状态
- 草稿
- 已提交
- 待审核
- 已通过
- 已驳回
- 已归档

## 5.4 导出任务状态
- 待生成
- 生成中
- 已完成
- 失败
- 已过期

## 5.5 协作入口状态
- 未启用
- 启用中
- 已过期
- 已撤销

# 6. 功能需求

## 6.1 登录与账号安全

### 功能点
- 账号密码登录
- 短信验证码登录
- 退出登录
- 修改密码
- 重置密码
- 首次登录强制修改密码
- 登录失败次数限制
- 异常登录提醒

### 规则
- 后台支持账号密码登录
- 可选短信验证码二次验证
- 第三方入口默认有效期 72 小时
- 被禁用账号不可登录
- 敏感角色变更后需重新登录生效

## 6.2 项目管理

- 新建项目
- 编辑项目
- 配置项目成员

- 项目负责人

- 结束日期

- 点位总数
- 已完成数量
- 待审核数量

- 项目编号唯一
