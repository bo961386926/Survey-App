# 青泓项目勘察信息采集与审核系统 - 需求文档

## 版本信息

- 当前版本：V5.1 汇总版
- 更新日期：2026-05-31
- 来源合并：PRD V3.2、移动端需求开发文档 Mobile v5.0、Web 端 UI v5.0 设计稿、SaaS 化改造建议
- 适用范围：PC 后台、移动采集端、第三方协作入口

## 1. 项目概述

### 1.1 项目定位

系统面向项目勘察与分散式/场景站点现场采集场景，覆盖概览工作台、地图视图、项目管理、任务中心、团队管理、点位管理、移动采集、动态模板、审核闭环、系统配置、消息协同、导出归档、第三方协作与审计留痕。

当前业务样板为“亳州入河排污口项目移动端勘察系统”，平台名称沿用“青泓项目勘察信息采集与审核系统”。

### 1.2 建设目标

- 统一现场采集标准，支持不同场景站点类型动态表单。
- 支持弱网、离线草稿、失败重试，降低现场数据丢失。
- 支持 GPS 定位、人工纠偏、纠偏轨迹追溯。
- 支持 PC 端地图视图、点位详情抽屉、筛选中心、卫星图/标准图切换和坐标纠偏。
- 建立提交、审核、驳回、重提、归档的闭环。
- 建立任务创建、分配、执行、进度动态、报告导出的任务闭环。
- 保留采集结果版本，驳回重提 100% 可追溯。
- 支持 PDF、Excel、CSV、ZIP 等归档导出。
- 支持团队成员、角色权限、组织架构、基础字典、系统参数和公告发布配置。
- 支持站内消息、公告通知、任务对话和发送日志。
- 支持第三方受限协作入口，保证授权范围与审计安全。

### 1.3 成功指标

- 草稿丢失率小于 0.5%。
- 驳回重提版本 100% 可追溯。
- PDF 报告生成成功率大于 99%。
- 第三方越权访问 100% 阻断并记录。
- 审核记录与关键操作 100% 留痕。

## 2. 角色与权限

### 2.1 角色

- 管理员：项目、标段、点位、模板、用户、角色、协作入口、全局导出、日志管理。
- 项目负责人：管理所属项目，配置成员、标段、模板绑定，查看进度与风险。
- 审核员：处理待审核记录，查看版本差异，生成 PDF，查看审核日志。
- 采集员：查看分配点位，定位纠偏，填写动态表单，保存草稿，提交审核。
- 外业主管：查看团队与任务看板，分配外业任务，跟踪任务进度，处理现场协同消息。
- 数据审计员：复核采集数据、查看审计日志、导出审计材料，不参与项目配置。
- 第三方协作人员：通过授权入口访问指定点位，协助填报或补充照片，不可审核、删除、全局导出。

### 2.2 权限原则

- 项目负责人仅能管理绑定项目。
- 审核员仅能查看被分配或授权的审核范围。
- 采集员仅能查看和采集分配点位。
- 第三方仅能访问协作入口授权范围。
- 敏感操作包括角色变更、导出、协作入口重置、点位作废，必须记录操作日志。
- 组织与权限管理支持按角色、部门、菜单访问权限和核心操作行为配置权限。
- 高危权限包括物理数据删除、数据批量导出、API 密钥管理、实时日志规则，需二次确认或审计记录。

## 3. 业务流程

### 3.1 主流程

1. 管理员创建项目、标段、成员与模板。
2. 按场景类型绑定模板版本。
3. 导入或新增点位，点位必须包含场景类型。
4. 分配点位给采集员。
5. 采集员开始采集，点位进入草稿中。
6. 移动端自动匹配模板版本，采集员定位、纠偏、拍照、填表。
7. 系统自动保存草稿，采集员提交审核。
8. 审核员通过或驳回。
9. 驳回后，系统复制被驳回版本生成新的草稿版本，原版本只读。
10. 通过后，系统自动生成单点位 PDF 并归档，点位纳入可导出范围。

### 3.2 模板流程

- 创建模板草稿。
- 配置字段、校验、联动、默认值、选项来源。
- 预览移动端效果。
- 发布模板版本。
- 在项目或标段下，将已发布模板版本与场景类型绑定。
- 已发布模板不可直接覆盖，修改必须生成新版本。

### 3.3 第三方协作流程

- 创建协作入口，设置授权点位、权限范围、有效期。
- 生成链接、小程序码或临时账号。
- 第三方在授权范围内查看、协助填报、上传图片。
- 系统记录访问与操作日志。
- 到期自动失效，也可手动撤销或重置 Token。

## 4. 状态机

### 4.1 项目状态

- 状态值：草稿(0)、进行中(1)、已暂停(2)、已完成(3)、已归档(4)。
- 流转规则：
  - 草稿 → 启动 → 进行中（建议检查标段/点位/采集员/模板就绪，不强制阻止）
  - 进行中 → 暂停 → 已暂停（采集员收到通知）
  - 已暂停 → 恢复 → 进行中
  - 进行中 → 完成 → 已完成（建议完成率100%，允许强制完成）
  - 已完成 → 归档 → 已归档（项目级操作，所有点位变为只读）
  - 已归档 → 取消归档 → 已完成（操作记录审计日志）
- 操作权限：
  - 草稿项目可编辑、可删除
  - 进行中/已暂停项目可编辑、不可删除
  - 已完成项目受限编辑、不可删除
  - 已归档项目只读、不可删除
- 前端操作入口：项目详情页头部根据当前状态动态显示可用操作按钮（启动/暂停/恢复/完成/归档/取消归档）

### 4.2 点位展示状态

点位状态是根据最新采集结果聚合生成的展示状态，不建议直接作为业务真实状态存储。

| 最新采集结果 | 点位展示状态 |
| --- | --- |
| 无采集结果 | 待采集 |
| 草稿 | 草稿中 |
| 已提交/待审核 | 待审核 |
| 已通过 | 审核通过 |
| 已驳回 | 驳回待修改 |
| 已归档 | 已归档 |
| 人工作废 | 作废 |

### 4.3 采集结果状态

- 草稿、已提交、待审核、已通过、已驳回、已归档。
- 每个采集结果按点位维护 `version_no`。
- 驳回后新增草稿版本，`version_no = 被驳回版本 + 1`，原版本只读。

### 4.4 其他状态

- 导出任务：待生成、生成中、已完成、失败、已过期。
- 协作入口：未启用、启用中、已过期、已撤销。
- 任务状态：待分配、进行中、审核中、已完成、逾期、已终止。
- 公告状态：草稿、定时发布、已发布、已撤回。
- 字典项状态：已启用、已禁用。
- 系统运行状态：运行中、维护中、异常。

## 5. 功能需求

### 5.1 PC 端全局框架

- 左侧主导航包括概览工作台、地图视图、项目管理、任务中心、团队管理、模板引擎、审核中心、系统管理。
- 顶部支持全局搜索，可搜索项目、任务、点位编号、人员、消息或文档。
- 顶部工具区提供消息通知、帮助、系统设置和当前账号入口。
- 系统需展示品牌 Logo、系统名称、版本标识和当前用户角色。
- 关键页面支持白底品牌蓝视觉规范，操作按钮、状态标签、图标、卡片保持统一。
- 系统管理可展开二级菜单，包括组织与权限、审计日志、系统参数、基础字典、消息通知中心等。

### 5.2 登录与账号安全

- 支持账号密码登录、短信验证码登录、退出登录、修改密码、重置密码。
- 移动端支持手机号 + 密码登录，手机号 11 位校验，密码 8-20 位且包含字母和数字。
- 注册需填写姓名、电话、所属部门并申请角色，后台审批通过后方可登录。
- 支持首次登录强制修改密码、登录失败次数限制、异常登录提醒。
- 第三方入口默认有效期 72 小时。
- 被禁用账号不可登录，敏感角色变更后需重新登录生效。

### 5.3 概览工作台

- 展示年度项目总数、当前活跃项目、已验收成果物、预警异常项等核心指标。
- 支持展示系统健康度、勘察资源分配率、项目逾期数量、完成率等运营指标。
- 支持勘察任务趋势图，可按月度、季度、年度切换。
- 支持实地勘察覆盖分布、云端同步状态和最近活动流。
- 最近活动流需展示成果物上传、审核通过、新成员加入、系统更新等事件，并支持进入历史活动。

### 5.4 项目与标段管理

- 支持新建、编辑、归档项目。
- 支持项目成员、标段、负责人、重点区域、完成率和审核积压查看。
- 支持按场景类型绑定模板版本。
- 项目编号唯一，已归档项目不可编辑。
- 管理员和项目负责人可修改项目配置。
- 项目列表需展示项目编号、项目名称/勘察区域、负责人、勘察进度、当前状态、更新时间和详情入口。
- 项目状态筛选至少包括全部、勘察中、待审核、已完成、已逾期。
- 新建项目弹窗字段包括项目名称、委托单位/客户、开始日期、预计结束日期、项目负责人、项目描述/备注。
- 项目详情需展示项目编号、项目名称、客户单位、项目周期、当前阶段、项目经理、进度详情、最近动态和主要利益相关者。
- 项目详情支持编辑项目、查看地图、添加成员、查看成员在线/离线状态、查看最近活动。
- 项目进度需展示总完成度、已完成点位、待处理点位、异常警报数量。

### 5.5 任务中心

- PC 端任务中心支持看板视图和列表视图切换。
- 任务看板按待分配任务、进行中任务、已完成任务分列展示，并显示每列数量。
- 任务总览需展示逾期任务、进行中、本周已完成等统计卡片。
- 支持按全部任务、待分配、进行中、审核中等状态筛选，并支持筛选、排序。
- 支持创建勘察任务，字段包括任务名称、所属项目、任务目标、具体地块编号、精度要求、传感器类型、优先级、截止日期、负责人和子任务。
- 任务详情需展示任务描述、基础属性、执行团队、作业进度流、测绘点位定位图层、勘察子任务清单。
- 任务详情支持编辑任务、分享任务、导出报告。
- 子任务可展示已完成、待开始、需现场控制点确认等状态，并支持“立即开始”。
- 作业进度流需记录任务分配、状态变更、数据同步、评论或动态回复。
- 任务消息可进入消息对话中心，与成员围绕任务沟通、发送附件和图片。

### 5.6 点位管理

- 支持 Excel 导入、手工新增、编辑、地图查看、批量分配、状态查看、作废、历史版本查看。
- 点位字段包括点位编号、名称、项目、标段、场景类型、经纬度、行政区、分配人、采集人、最新版本号、异常标签。
- 作废点位不可再采集，已归档点位不可再产生新版本。
- 点位批量导入弹窗包括下载导入模板、上传数据文件、数据处理规则三段流程。
- 批量导入支持 `.xlsx`、`.xls`、`.csv`，单文件大小不超过 50MB。
- 导入冲突处理支持跳过重复项和覆盖重复项；重复判断以点位编号/名称等唯一规则为准。
- 导入完成需返回成功数、失败数、冲突数和失败原因明细。

### 5.7 地图视图与点位详情

- PC 端地图视图支持项目选择、点位状态筛选、勘察人员查看、点位图层和底图切换。
- 点位状态筛选至少包括待处理、进行中、已完成、异常。
- 地图支持卫星图和标准地图切换，支持缩放、定位、图层控制。
- 点位以状态颜色或图标区分，点击点位打开点位详情抽屉。
- 点位详情抽屉展示点位编号、当前勘察状态、优先级、经纬度、人工纠偏入口、勘察基础属性、现场勘察附件和提交审核按钮。
- 勘察基础属性包括勘察设备、采样频率、精度阈值、最后同步时间等。
- 人工纠偏需记录纠偏前后坐标、纠偏人、纠偏时间和偏移距离。

### 5.8 动态模板引擎

- 字段类型：Input、Number、Select、MultiSelect、Radio、Checkbox、Switch、Date、Image、Location、Textarea。
- 支持字段拖拽排序、分组布局、必填校验、格式校验、范围校验、联动显隐、联动清空、默认值设置。
- Image 字段支持“仅允许相机拍摄”配置。
- 选项来源支持静态选项和系统数据字典、行政区划等动态数据源。
- 历史采集结果绑定提交时使用的模板版本。
- PC 端模板引擎需提供组件库、表单画布、属性配置三栏布局。
- 组件库至少包括单行输入、多行输入、单选框、多选框、图片上传、位置打点。
- 属性配置支持字段标签、是否必填、选项管理、逻辑联动和高级联动引擎。
- 模板编辑顶部支持问卷编辑、样式设置、发布预览、搜索、预览和发布问卷。
- 系统需提供智能诊断，识别模板逻辑冲突并给出提示。

### 5.9 移动端采集

- 底部主导航：工作台/看板、点位/作业、地图、勘查、我的；地图作业中心是现场高频入口。
- 地图作业中心支持高德/卫星图、自动定位、缩放旋转、点位预览、人工纠偏和开始数据录入。
- 今日看板展示已完成、进行中、待同步、逾期等指标及最近任务。
- 作业列表支持距离优先、状态、优先级筛选。
- 动态表单根据点位场景类型自动加载模板版本。
- 用户输入每 30 秒自动本地暂存，退出后可恢复。
- 图片上传按模板配置调用相机或允许相册，原图上传，OSS 展示时拼接水印。
- 水印包含采集人、拍摄时间、经纬度。
- 弱网提交失败需保留草稿和失败原因，网络恢复后可重试。
- 并发控制：打开点位时获取版本号，保存/提交时服务端校验，不一致则拒绝并提示刷新。

### 5.10 审核中心

- 支持待审核列表、筛选、单条审核、批量通过、地图位置校验、图片水印校验、版本差异查看、驳回原因模板。
- 审核结果仅支持通过/驳回，驳回必须填写原因。
- 仅最新版本允许审核，历史版本只读。
- 预留转审能力，后端日志可记录 `transfer` 操作。
- PC 端审核中心左侧展示待审列表，支持全部、高优先级、异常件筛选。
- 审核详情展示项目/点位标题、位置、版本号、历史追溯、版本变更历史、采集数据对比、照片水印查验和审核意见。
- 版本对比需高亮字段差异，展示历史值、当前提交值和差异数量。
- 照片查验需展示原图、坐标、时间、设备信息和 AI 查验结果。
- 驳回弹窗需支持快捷短语、详细理由、附件上传和确认驳回。
- 驳回附件支持图片、PDF，单次上传大小不超过 10MB。

### 5.11 导出与报表

- 支持点位清单、审核结果、单点位 PDF、批量 PDF ZIP、CSV 导出。
- 审核通过后自动生成单点位 PDF 并归档，支持手动重新生成。
- 大批量导出必须走异步任务。
- 导出行为记录日志，下载链接可过期。
- 第三方不可全局导出。
- 数据全局导出弹窗支持选择导出范围：当前筛选范围或全部项目数据。
- 导出格式支持 Excel 表格、PDF 报告包、原始水印图。
- 文件命名规则可由项目名称、排污口编号、导出日期等字段组合配置，并提供文件名预览。
- 审计日志支持导出 Excel 和 PDF。

### 5.12 团队管理与用户档案

- 团队管理需展示总成员数、活跃团队、平均效能评分、团队绩效趋势和成员详情列表。
- 成员详情列表需展示成员姓名、工号、所属单位、职位角色、联系方式、活跃任务、综合评分和更多操作。
- 支持内勤/外业等成员类型筛选，支持按部门筛选。
- 添加团队成员采用分步流程：基本信息、角色与权限、完成。
- 添加成员基本信息包括姓名、工号、手机号、邮箱、所属部门。
- 系统需展示实名认证、数据加密、配额限制等安全提示。
- 用户档案需展示头像、姓名、职位、员工号、认证状态、地区、联系信息、办公地点、勘察积分、效率指数、质检通过率、专业资质认证和工作动态。
- 编辑个人资料支持头像上传、姓名、工号、手机号、所属部门、个人简介编辑；手机号等敏感字段可设为不可修改。

### 5.13 系统管理、字典与参数

- 基础字典支持字典分类、字典项列表、状态统计、批量导入、新增字典项、启用/禁用和分页。
- 字典分类至少包括行政区划、场景类型、河道信息、施工阶段、点位分级、监测指标、审批状态。
- 系统参数配置包括基本设置、运行配置、版本管理、安全设置。
- 基本设置包括系统名称、页脚版权信息、品牌 Logo 上传。
- 运行配置包括偏移预警阈值和自动保存草稿间隔。
- 版本管理包括最新更新日期、更新日志、强制更新开关。
- 安全设置包括密码过期周期、多设备登录限制。
- 配置变更必须记录配置日志。

### 5.14 组织权限与审计日志

- 组织与权限管理支持角色体系、组织架构树、菜单访问权限和核心操作行为配置。
- 角色卡片需展示角色描述、标签、成员数量和选中状态。
- 菜单访问权限支持按模块展开配置，例如全局概览工作台、项目管理中枢、系统底层设置等。
- 核心操作行为支持独立开关，例如物理数据删除、数据批量导出、API 密钥管理、实时日志规则。
- 配置影响范围需展示活跃用户数、敏感操作数和最后修改时间。
- 审计日志需支持按操作人、模块类型、操作时间范围筛选。
- 审计日志列表字段包括操作人、所属单位、操作模块、操作内容、操作 IP、发生时间、状态。
- 日志详情可展开展示驳回原因、失败原因等上下文。
- 审计日志支持分页、导出 Excel、导出 PDF。

### 5.15 消息、公告与协同

- 用户管理支持新增、编辑、启停、角色绑定、项目范围绑定、重置密码、账号日志。
- 日志类型包括登录、审核、导出、权限变更、协作访问、风控拦截、纠偏操作。
- 消息类型包括待审核提醒、项目延期预警、模板发布通知、导出完成通知、协作到期提醒、风控告警。
- 触达渠道包括系统内消息中心、App Push、小程序服务通知。
- 消息中心需支持消息列表、未读筛选、消息详情、收藏、删除和更多操作。
- 消息详情可承载任务分配、公告、系统简报等内容，并支持附件下载和底部快速回复。
- 消息对话中心支持最近联系人、在线状态、聊天记录、图片/文件附件、下载附件、发送消息。
- 消息通知中心看板展示今日发送总量、平均阅读率、异常推送率。
- 推送配置支持场景化规则，字段包括业务场景、推送方式、接收对象、生效状态和编辑规则。
- 推送方式至少包括 App Push、短信、站内消息。
- 支持消息模板维护、发送历史、实时发送流水和发送日志。
- 公告发布中心支持公告列表、搜索、新建公告、编辑、删除、状态查看和公告详情预览。
- 公告字段包括公告标题、类型、发布人、发布时间、状态、受众范围和正文内容。
- 公告类型至少包括作业规范、维护提醒、系统通知。
- 公告支持草稿、定时发布、已发布等状态。

## 6. 非功能需求

- 性能：列表分页，地图点位聚合，表单字段懒加载，大文件导出异步化。
- 可用性：草稿自动恢复，提交失败可重试，图片上传支持单张重试，关键操作有明确反馈。
- 安全：全站 HTTPS，密码加密存储，文件下载鉴权，敏感操作留痕。
- Token：JWT 载荷区分 `login_type=internal/collab`，协作 Token 包含 `collab_entry_id` 并严格校验接口权限。
- 兼容性：iOS 12+、Android 8+、微信小程序最新版、Chrome/Edge 最新两个主版本。
- SaaS 多租户：已实现行级隔离模式（Shared Table），MyBatis Plus 拦截器自动追加 `tenant_id` 条件，业务层零改动。详见设计文档第9章。

## 7. 研发验收重点

- 点位场景类型与模板版本绑定后，移动端表单自动匹配。
- 驳回重提生成新版本，原版本数据只读保留。
- 并发编辑时，后提交者收到版本冲突提示。
- 第三方 Token 无法调用审核、删除、全局导出等越权接口。
- 纠偏操作记录可在地图中心查看轨迹。
- Image 字段“仅允许相机拍摄”配置实时生效。
