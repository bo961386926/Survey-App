# 青泓项目勘察信息采集与审核系统 - 测试文档

## 版本信息

- 当前版本：V5.0 汇总版
- 更新日期：2026-05-29
- 来源合并：测试用例清单 V1、部署质量保障方案、PRD V3.2 验收标准、Mobile v5.0 重点链路

## 1. 测试范围

- PC 后台：认证、项目、标段、点位、模板、审核、地图、导出、用户角色、字典、日志、消息。
- 移动端：登录、工作台、点位列表、点位地图、定位纠偏、动态表单、图片上传、草稿、提交、消息、我的。
- 第三方协作：入口有效期、授权范围、越权拦截、访问日志。
- 非功能：权限安全、并发控制、弱网恢复、文件导出、部署前检查。

## 2. 测试策略

- 单元测试：后端 Service、工具类、权限判断、版本生成、模板匹配。
- 集成测试：认证流程、核心业务接口、数据库状态流转。
- E2E 测试：登录到采集提交、审核通过/驳回、导出、第三方入口。
- 兼容性测试：iOS 12+、Android 8+、微信小程序最新版、Chrome/Edge 最新两个主版本。
- 弱网测试：断网、超时、图片上传失败、恢复后重试。
- 安全测试：越权访问、Token 过期、协作 Token 调用内部接口、禁用账号。

## 4. 非功能测试

### 4.1 弱网与离线

- 断网填写表单，草稿不能丢失。
- 图片上传失败支持单张重试。
- 恢复网络后可继续提交。
- 本地草稿与服务端草稿冲突时给出明确提示。

### 4.2 性能

- 列表分页响应稳定。
- 地图大量点位启用聚合，缩放和筛选不卡顿。
- 动态表单字段较多时支持懒加载或分组加载。
- 大文件导出不阻塞主请求。

### 4.3 安全

- 未登录访问业务接口返回未授权。
- 普通用户无法访问管理员接口。
- 第三方 Token 无法调用内部后台接口。
- 文件下载链接过期后不可访问。
- 敏感操作必须产生操作日志。

## 5. 部署前检查

### 5.1 后端

- 应用启动成功。
- 数据库迁移或初始化脚本执行成功。
- 健康检查接口可访问。
- Swagger/Knife4j 文档可访问。
- 单元测试和集成测试通过。
- 关键接口：登录、项目列表、点位列表、采集详情、保存草稿、提交审核、审核通过、导出任务。

### 5.2 前端

- PC 后台构建通过。
- 移动端构建通过。
- 环境变量指向正确 API 地址。
- 路由可访问，刷新不 404。
- 登录态过期后可正确跳转登录页。

### 5.3 E2E 主链路

1. 管理员登录 PC 后台。
2. 创建项目、标段、模板、点位。
3. 绑定排口类型与模板版本。
4. 分配点位给采集员。
5. 采集员移动端登录并提交采集。
6. 审核员审核通过。
7. 系统生成 PDF，并在导出中心可下载。
8. 操作日志和消息提醒可查询。

## 6. 验收准入

- P0 功能用例全部通过。
- 安全和越权用例全部通过。
- 弱网草稿和并发冲突用例全部通过。
- 关键 E2E 主链路全部通过。
- 未解决缺陷需标注严重级别、影响范围、规避方式和计划修复时间。
