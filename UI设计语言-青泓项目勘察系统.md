# 1. 设计理念

## 1.1 核心原则

```
简洁 · 清晰 · 深度
Simplicity · Clarity · Depth
```

### 设计哲学
- **内容优先**：界面服务于内容，去除一切不必要的装饰
- **层次分明**：通过毛玻璃层级建立清晰的信息架构
- **克制优雅**：少即是多，用留白和阴影创造呼吸感
- **一致体验**：PC端与移动端保持统一的设计语言

## 1.2 Apple设计美学参考

| Apple设计原则 | 本系统应用 |
|--------------|-----------|
| 清晰的视觉层次 | 毛玻璃卡片层级 + 阴影深度 |
| 直观的操作反馈 | 微动效 + 状态色 + 触觉反馈 |
| 内容优先的布局 | 大留白 + 卡片式信息分组 |
| 一致的交互模式 | 统一的组件库 + 标准导航模式 |

# 5. 毛玻璃效果规范（Glassmorphism）

## 5.1 毛玻璃层级体系

```
┌─────────────────────────────────────────┐
│  Layer 3: 弹窗/抽屉 (z-index: 1000+)    │  ← 最强毛玻璃
│  ┌───────────────────────────────────┐  │
│  │ Layer 2: 顶部导航/底部Tab (z: 100) │  │  ← 中等毛玻璃
│  │                                   │  │
│  │  ┌─────────────────────────────┐  │  │
│  │  │ Layer 1: 卡片 (z: 1)        │  │  │  ← 轻量毛玻璃
│  │  │                             │  │  │
│  │  │  内容区域                    │  │  │
│  │  └─────────────────────────────┘  │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

## 5.2 毛玻璃实现参数

### 5.2.1 卡片毛玻璃（轻量）

```css
.glass-card {
  background: rgba(255, 255, 255, 0.65);
  backdrop-filter: blur(20px) saturate(180%);
  -webkit-backdrop-filter: blur(20px) saturate(180%);
  border: 1px solid rgba(255, 255, 255, 0.4);
  border-radius: 16px;
  box-shadow: 
    0 2px 8px rgba(0, 0, 0, 0.04),
    0 8px 24px rgba(0, 0, 0, 0.06);
}
```

### 5.2.2 导航栏毛玻璃（中等）

```css
.glass-navbar {
  background: rgba(255, 255, 255, 0.72);
  backdrop-filter: blur(40px) saturate(200%);
  -webkit-backdrop-filter: blur(40px) saturate(200%);
  border-bottom: 1px solid rgba(255, 255, 255, 0.5);
  box-shadow: 0 1px 4px rgba(0, 0, 0, 0.04);
}
```

### 5.2.3 弹窗毛玻璃（强）

```css
.glass-modal {
  background: rgba(255, 255, 255, 0.85);
  backdrop-filter: blur(60px) saturate(220%);
  -webkit-backdrop-filter: blur(60px) saturate(220%);
  border: 1px solid rgba(255, 255, 255, 0.6);
  border-radius: 20px;
  box-shadow: 
    0 24px 48px rgba(0, 0, 0, 0.12),
    0 0 0 1px rgba(0, 0, 0, 0.04);
}
```

### 5.2.4 深色模式毛玻璃

```css
.glass-card-dark {
  background: rgba(30, 30, 30, 0.65);
  backdrop-filter: blur(20px) saturate(180%);
  border: 1px solid rgba(255, 255, 255, 0.12);
  border-radius: 16px;
  box-shadow: 
    0 2px 8px rgba(0, 0, 0, 0.2),
    0 8px 24px rgba(0, 0, 0, 0.3);
}
```

## 5.3 毛玻璃使用场景

| 场景 | 类型 | 背景要求 |
|------|------|----------|
| 数据卡片 | 轻量毛玻璃 | 需要渐变或图片背景 |
| 顶部导航 | 中等毛玻璃 | 页面滚动时内容透出 |
| 弹窗/对话框 | 强毛玻璃 | 遮罩层背景模糊 |
| 底部操作栏 | 中等毛玻璃 | 移动端底部固定 |
| 侧边抽屉 | 强毛玻璃 | 内容层模糊 |
| 地图浮层 | 轻量毛玻璃 | 地图内容透出 |

## 5.4 毛玻璃注意事项

⚠️ **性能优化**
- 避免大面积使用 backdrop-filter（影响GPU）
- 导航栏/底部Tab使用 will-change: transform
- 移动端适当降低 blur 值（20px → 16px）

⚠️ **可访问性**
- 确保文字对比度 ≥ 4.5:1
- 毛玻璃上文字使用 Gray-800/900
- 禁用状态需要更明显的视觉区分

   - 文字清晰可读
   - 图标表意明确
